Project Overview

Gene’s Drives is an educational website that explains the science, ethics, and real-world applications of gene drives and in-vivo genetic therapies. The site aims to inform a general audience about these cutting-edge medical technologies, highlighting both their potential and ethical implications.

The website is frontend-only, built using HTML, CSS, and Bootstrap, and focuses on accessibility, readability, and a clean, modern scientific aesthetic.

Aim

To provide clear and engaging information about:

How gene drives work

Their potential impact on medicine and public health

Ethical and cultural considerations

Current breakthroughs and applications

Pages

Home: Overview of gene drives, their mechanics, and significance for public health.

About: Ethical considerations, cultural perspectives, recent breakthroughs, and real-world applications.

Contact: Newsletter signup form and contact option for users to engage with the project.

Features

Dashboard layout: Clean, card-style sections for information display

Hero sections with responsive images: Illustrative and engaging visuals

Floating “Return-to-Top” functionality (optional JavaScript or CSS enhancement)

Mobile-friendly design using Bootstrap responsive classes

Interactive elements: Form validation on contact/newsletter form

Theme & Design

Color Scheme: Navy blue (#0a1a33), white, and grey (#f2f2f5)

Typography: Modern system fonts for readability

Visual Style: Clean, educational, and science-themed with subtle shadows and rounded imagery

Technology Stack

HTML5

CSS3

Bootstrap 5.3

JavaScript (for form validation and interactive alerts)

Fontisto icons for navigation
